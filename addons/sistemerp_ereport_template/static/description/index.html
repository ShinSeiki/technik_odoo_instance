Multiple Invoice Template

This application helps to generate dynamic PDF reports.

Basic Features
  • Allows you to choose multiple report layouts.
  • Let's make a color combination.
  • You can assign a special report for each customer
After installation of this application below configuration will appear in Settings.
Libraries (Python) for application installation

img2pdf

                  sudo apt-get install python3-img2pdf
                              OR
                  sudo pip3 install img2pdf

fpdf

                  sudo apt-get install python3-fpdf
                              OR
                  sudo pip3 install fpdf
